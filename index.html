ruby-xtract by christoph-buente

An FFI wrapper for libxtract (because you always wanted to do audio processing with ruby)

Dependencies

  sudo port install fftw-3
  sudo port install libffi
  sudo gem install ffi

Install

cd path/to/your/src/directory
wget http://downloads.sourceforge.net/sourceforge/libxtract/libxtract-0.4.7.tar.gz
tar xvfz libxtract-0.4.7.tar.gz
cd libxtract-0.4.7
./configure --prefix=/opt/local \
            --enable-fft \
            --with-fftw3-dir=/opt/local/include \
            --libdir=/opt/local/lib
make
sudo make install

sudo gem -a http://gems.github.com
sudo gem install ruby-xtract
